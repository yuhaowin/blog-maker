### 颁发证书

```bash
acme.sh --issue --dns dns_dp -d yuhaowin.com -d *.yuhaowin.com
```

### 部署证书到七牛云

```bash
export QINIU_AK="_woYoRMBoiC_qRYOmaB4ukjAOS7tPYhrzJS-wf82"
export QINIU_SK="WOtu_RHHWgUpSyDj-GSz98tl1Qo_5Kg89fsr6azm"
export QINIU_CDN_DOMAIN=".yuhaowin.com"
acme.sh --deploy -d yuhaowin.com --deploy-hook qiniu
```

### 部署证书到本地 nginx

```bash
acme.sh --install-cert -d yuhaowin.com \
--key-file       /home/ubuntu/app/nginx/cert/yuhaowin.com/yuhaowin.com.key  \
--fullchain-file /home/ubuntu/app/nginx/cert/yuhaowin.com/fullchain.cer \
--reloadcmd     "docker restart nginx"
```
